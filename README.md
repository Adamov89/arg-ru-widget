# Arg Ru Widget 🇦🇷

A lightweight iPhone widget for learning Argentine Spanish (Buenos Aires / Rioplatense vocabulary).

The widget displays:

* one Argentine Spanish word
* Russian translation
* word counter

The word changes automatically every 15 minutes.

No account required.
No subscription required.
No coding experience required.

---

## Features

* 5000 Argentine Spanish words
* Buenos Aires focused vocabulary
* Automatic rotation every 15 minutes
* Works directly on the iPhone Home Screen
* Progress tracking
* Dictionary updates can be distributed through GitHub

## Installation

### 1. Install Scriptable

Download Scriptable from the App Store.

### 2. Create a new script

Open Scriptable.

Tap:

`+`

Create a new script.

Name it:

`Arg Ru`

### 3. Paste the code

Copy the code from ArgRu.js and paste it into Scriptable
Save.

### 4. Run once

Tap the ▶ button.

You should see a preview with a Spanish word and its Russian translation.

### 5. Add the widget

On the iPhone Home Screen:

1. Long press on an empty area.
2. Tap `+`.
3. Search for `Scriptable`.
4. Select the small widget.
5. Add Widget.

### 6. Connect the script

Long press the widget.

Choose:

`Edit Widget`

Select:

`Arg Ru`

Done.

---

## Updating the Dictionary

The dictionary is hosted on GitHub.

When the dictionary file is updated, users automatically receive the new vocabulary without reinstalling the widget.

No manual download is required.
